# Node.js Project

## Description

This is a Node.js project using Express, JWT authentication, and **PostgreSQL** for backend operations. It includes authentication, category management, rate limiting, and security features.

## Tech Stack

* Node.js
* Express
* JWT Authentication
* PostgreSQL
* Helmet
* CORS
* express-rate-limit

## Features

* User registration and login (JWT-based)
* Profile management
* Category CRUD operations
* PostgreSQL database integration
* Rate limiting for sensitive endpoints
* Helmet for security headers
* CORS configuration
